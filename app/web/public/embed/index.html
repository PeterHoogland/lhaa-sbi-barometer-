Integratie-code

<div id="sbi-banner"></div>
<script src="https://nationalestressindex.be/embed/banner.js" defer></script>
<script>
  window.addEventListener('load', () => SBI.mount({
    target: '#sbi-banner',
    apiUrl: 'https://nationalestressindex.be/data/latest.json',
    brand: 'Les Hautes Alpes',
    ctaUrl: 'https://www.hautes-alpes.net'
  }));
</script>

Wat het script doet

  • Haalt eens per pagina-load het huidige SBI-signal op
  • Render banner-variant op basis van conditie-niveau 1-5
  • Bij CN 1, 2, 5: geen banner, stil uitgeschakeld, geen visuele ruis
  • Brand-safety-override automatisch (CN 5 onderdrukt commerciële uitnodiging)
  • Geen cookies, geen tracking, geen externe dependencies

Naleving stijlgids (doc 09)

  • ✓ Copy is hard-gecodeerd in het script, de abonnee kan deze niet wijzigen
  • ✓ Brand-naam en CTA-URL zijn de enige configureerbare velden
  • ✓ Bij brand-safety-vlag: automatische onderdrukking
  • ✓ Methodologie-link altijd zichtbaar in de banner-footer
